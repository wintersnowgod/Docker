## homarr
- It is a custom Homepage that can be customized to show the app shortcuts, dns (like pihole) stats, docker status and much more!  
- To use it just start the container using the command:-  
`docker compose -f homarr.yml -p homarr up -d`  
- To stop the container  
`docker compose -f homarr.yml -p homarr down`  
- You can access the homarr homepage now in  
http://localhost:7575    
- For any additional info refer to  
https://homarr.dev/docs/getting-started/installation/docker
https://github.com/homarr-labs/homarr
