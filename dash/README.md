## dash
- It is a service to show your system resource consumption.  
- To use it just start the container using the command:-  
`docker compose -f dash.yml -p dash up -d`  
- To stop the container  
`docker compose -f dash.yml -p dash down`  
- You can access the dash homepage now in  
http://localhost:3001  
- For any additional info refer to  
https://getdashdot.com/docs/installation/docker-compose  
https://github.com/MauriceNino/dashdot
